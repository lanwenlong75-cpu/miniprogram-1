# 项目背景与架构

## Project Overview
输入冰箱里的食材，秒出能做的菜谱

## Architecture

### Pattern
小程序原生

### UI Architecture
页面使用 WXML + WXSS，组件化使用 Component 构造器

### Data Layer
使用 wx.cloud 云开发数据库，app.globalData 做全局状态

### Dependency Injection
无依赖注入，通过 getApp() 获取全局实例

### Concurrency / Async Model
小程序单线程模型，异步使用 Promise/callback

## Key Dependencies
微信云开发 wx-server-sdk ~2.4.0
